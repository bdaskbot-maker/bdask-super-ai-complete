================================================================================
                            TROUBLESHOOTING
================================================================================

Issue: Build fails
Solution:
  1. Check logs: vercel logs --tail
  2. Verify dependencies: pnpm install
  3. Check environment variables
  4. Ensure Node.js >= 18.0.0

Issue: API returns 500 error
Solution:
  1. Verify NVIDIA_API_KEY is set
  2. Check API key is valid
  3. Check rate limits aren't exceeded
  4. Review error logs

Issue: Slow response time
Solution:
  1. Check NVIDIA API status
  2. Monitor network latency
  3. Verify model selection
  4. Consider upgrading instance

Issue: Models not appearing
Solution:
  1. Verify NVIDIA_API_KEY is set
  2. Check /api/agent/health endpoint
  3. Refresh browser (Ctrl+F5)
  4. Check browser console for errors
